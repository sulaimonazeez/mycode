Attributes of HTML Img tag

The src and alt are important attributes of HTML image tag. All attributes of HTML image tag are given below.

1) src

It is a necessary attribute that describes the source or path of the image. It instruct the browser where to look for the image on the server.

2) alt

< The alt attributes defines an alternate text for the image, if it can't be display. The value of the alt attribute describe the image in words. The alt atttribute is considered good for SEO propective.

3) width

It is an optional atttribute which used to specify the width to display the image. It is not recommended now. You should apply CSS in place of width attribute.

4) height

It h3 the height of the image. The HTML height attribute also supports iframe, image and object elements. It is not recommended now. You should apply CSS in place of height attribute.
