HTML Heading

A HTML heading or HTML h tag can be define as a title or a subtitle which you want to display on the webpage. When you place the text within the heading tag < h 1 > ... , it is displayed on the browser in the bold format and size of the text depend on the number of heading.

There are six different HTML heading which are define with the < h1 > to < h6 > tags from highest level h1 (main heading) to the least level h6 (least important heading).

h1 is the largest heading tag and h6 is the smallest one. So h1 is used for most important heading and h6 is used for least important.

Heading in HTML helps the search engine to understand and index the structure of web page.
Note: The main keyword of the whole of a webpage should be display by h1 tag
