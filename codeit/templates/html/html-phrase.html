HTML Phrase

The HTML phrase tag are special purpose tags, which define the structural meaning of a block of text or semantics of text. Following is the list of phrase tags some of which we have alreary discussed in HTML Formatting.

  • Abbreviation tag: < abbr >
  • Acronym tags: < acronym > (not supported in HTML5)
  • Marked tag < mark >
  • strong tag: < strong >
  • Emphasized tag: < em >
  • definition tag: < dfn >
  • Quoting tag: < blockquote >
  • Short quote tag: < q >
  • Code tag < code >
  • keyboard tag: < kbd >
  • Address tag: < address >
