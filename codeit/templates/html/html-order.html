HTML Ordered List | HTML Number List

HTML Ordered List or Numbered List displays element numbered format. The HTML ol tags is used for ordered list to represent items either in numerical order format or alphabetical order format, or any format where an order is emphasized. There can be different types of numbered list:

  • Numeric Number (1,2,4)
  • Captial Roman Number ( I II III)
  • Small Roman Number (i ii iii)
  • Small Alphabet (a,b,c)
  • Capital Alphabet (A, B, C)
