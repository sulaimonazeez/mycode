HTML Introduction

What's HTML5?

HTML stands for Hyper Text MarkUp Language, it's easy and fun to learn. HTML describe the structure of the web page. HTML5 is the fifth and current major version of the html standard

Why learn HTML5?

it's essential to learn HTML if you want to build web sites, you can't build one if you don't know HTML because it's one of the prerequisite in learning other language used for web developement.

Building blocks

An HTML document consist of it's basic building block which are:

  • Tags:

    An HTML tag sorrounds the content and apply meaning to it. it's written between < and > brackets.

  • Attribute:

    An attribute in HTML provides extra information about the element, and it is applied within the start tag. An HTML attribute contain two fields: name and value

Syntax

  • Elements:

    An HTML element is an individual component of an html file. In an HTML file, everything is written within tags as termed as HTML elements.

HTML Attribute

  • HTML Attribute are special words which provide additional information about the element or attribute are the modifier of the HTML element.
  • Each element or tag can have attribute, which define the behavior of that element.
  • Attribute should always be applied with it's name and value pair.
  • The Attributes name and value are case sensitive, and it's recommend by W3C that it should be written in Lowercase only.
  • You can add multiple attributes in one HTML element, but need to give space between two attributes
